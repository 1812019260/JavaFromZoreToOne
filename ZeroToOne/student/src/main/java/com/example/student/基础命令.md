# Java基础语法详解

## 1. Java程序结构

一个基本的Java程序包含以下部分：

```java
// 包声明（可选）
package com.example;

// 导入语句（可选）
import java.util.Scanner;

// 类声明
public class ClassName {
    // 主方法 - 程序入口点
    public static void main(String[] args) {
        // 程序代码
    }
}
```

## 2. 变量和数据类型

### 2.1 基本数据类型

| 数据类型 | 说明       | 默认值   | 范围                                                    |
| -------- | ---------- | -------- | ------------------------------------------------------- |
| byte     | 8位整数    | 0        | -128 到 127                                             |
| short    | 16位整数   | 0        | -32,768 到 32,767                                       |
| int      | 32位整数   | 0        | -2,147,483,648 到 2,147,483,647                         |
| long     | 64位整数   | 0L       | -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807 |
| float    | 32位浮点数 | 0.0f     | 精度约为6-7位小数                                       |
| double   | 64位浮点数 | 0.0d     | 精度约为15位小数                                        |
| boolean  | 布尔值     | false    | true 或 false                                           |
| char     | 16位字符   | '\u0000' | 0 到 65,535                                             |

## 3. 输入输出

### 3.1 输出语句

```java
// 打印并换行

// 格式化输出
System.out.printf("年龄是: %d, 身高是: %.2f\n", age, height);
```

## 5. 控制流程

### 5.1 条件语句

#### if语句

```java
if (条件) {
    // 条件为真时执行的代码
} else if (另一个条件) {
    // 另一个条件为真时执行的代码
} else {
    // 所有条件都为假时执行的代码
}
```

#### switch语句

```java
switch (变量) {
    case 值1:
        // 代码块
        break;
    case 值2:
        // 代码块
        break;
    default:
        // 默认代码块
        break;
}
```

### 5.2 循环语句

#### for循环

```java
for (初始化; 条件; 更新) {
    // 循环体
}

// 示例
for (int i = 0; i < 10; i++) {
    System.out.println("第" + i + "次循环");
}
```

#### while循环

```java
while (条件) {
    // 循环体
    // 更新条件
}

// 示例
int count = 0;
while (count < 5) {
    System.out.println("count = " + count);
    count++;
}
```

#### do-while循环

```java
do {
    // 循环体
    // 更新条件
} while (条件);
```

## 6. 数组

### 6.1 一维数组

```java
// 声明数组
int[] numbers;
String[] names;

// 创建数组
numbers = new int[5]; // 创建长度为5的整数数组
names = new String[]{"张三", "李四", "王五"};

// 访问数组元素
numbers[0] = 10; // 给第一个元素赋值
int firstNumber = numbers[0]; // 获取第一个元素的值

// 遍历数组
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

// 增强for循环
for (int num : numbers) {
    System.out.println(num);
}
```

## 7. 方法

### 7.1 方法定义

```java
修饰符 返回类型 方法名(参数列表) {
    // 方法体
    return 返回值; // 如果返回类型不是void
}

// 示例
public static int add(int a, int b) {
    return a + b;
}

public static void printMessage(String message) {
    System.out.println(message);
}
```

### 7.2 方法调用

```java
// 调用静态方法
int result = add(5, 3);
printMessage("Hello World");

// 调用实例方法（需要创建对象）
ClassName obj = new ClassName();
obj.instanceMethod();
```

## 8. 字符串操作

```java
String str1 = "Hello";
```

## 9. 异常处理

```java
try {
    // 可能出现异常的代码
    int result = 10 / 0;
} catch (ArithmeticException e) {
    // 处理特定异常
    System.out.println("除零错误: " + e.getMessage());
} catch (Exception e) {
    // 处理其他异常
    System.out.println("发生错误: " + e.getMessage());
} finally {
    // 无论是否发生异常都会执行的代码
    System.out.println("清理资源");
}
```

## 10. 完整示例程序

```java
import java.util.Scanner;

public class JavaBasicSyntax {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("Java基础语法示例程序");
        
        // 变量声明和赋值
        String name;
        int age;
        double height;
        
        // 输入
        System.out.print("请输入您的姓名: ");
        name = scanner.nextLine();
        
        System.out.print("请输入您的年龄: ");
        age = scanner.nextInt();
        
        System.out.print("请输入您的身高(米): ");
        height = scanner.nextDouble();
        
        // 输出
        System.out.println("姓名: " + name);
        System.out.println("年龄: " + age + "岁");
        System.out.printf("身高: %.2f米\n", height);
        
        // 条件语句
        if (age >= 18) {
            System.out.println("您已成年");
        } else {
            System.out.println("您未成年");
        }
        
        // 循环示例
        System.out.print("数字0到4: ");
        for (int i = 0; i < 5; i++) {
            System.out.print(i + " ");
        }
        System.out.println();
        
        // 方法调用
        int sum = add(10, 20);
        System.out.println("10 + 20 = " + sum);
        
        scanner.close();
    }
    
    // 自定义方法
    public static int add(int a, int b) {
        return a + b;
    }
}
```
